# Habesha Hub — Task Overview

> Ethiopian Business Directory for the Middle East.
> Built on Directory SaaS boilerplate (NestJS + Prisma + React + TypeScript).
> Starter tasks (01-27 backend, 28-48 frontend) are already complete.

## Phase 1: Domain Customization — Backend (Tasks 50–55)

| # | Task | Description |
|---|------|-------------|
| 50 | Schema & Seed — Habesha Hub Domain | Add country/city, multi-category, WhatsApp, business hours, deals; seed categories and demo data |
| 51 | Multi-Category Tagging | Many-to-many business ↔ category, search by multiple categories, filter API |
| 52 | Country & City Support | Country/city on provider profiles, location-based search, city autocomplete |
| 53 | Business Verification System | Verification request flow, admin approval, verified badge, trust score |
| 54 | WhatsApp & Contact Integration | WhatsApp click-to-chat, phone, social links, contact analytics |
| 55 | Deals & Promotions Module | Business deals/offers with expiry, featured deals, deal categories |
