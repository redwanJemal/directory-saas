# Task 70: Mobile App — Expo

## Summary
Build a consumer mobile app with Expo for business discovery, reviews, and saved businesses. Reuse the existing Expo scaffold in apps/mobile.

## Required Changes
- Home screen: country/city selector, category grid, featured businesses
- Search: text search + filters (category, city, rating)
- Business profile: full details, gallery, reviews, WhatsApp button
- Saved businesses: bookmark businesses for later
- Reviews: write reviews from mobile
- Push notifications: new deals in your city, new businesses
- Auth: client login/register
- Deep linking: open business profiles from shared links

## Acceptance Criteria
- [ ] Home screen with categories and featured businesses
- [ ] Search with filters
- [ ] Business profile with WhatsApp button
- [ ] Save/bookmark businesses
- [ ] Push notifications for deals
